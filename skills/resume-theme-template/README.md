# 📄 简历主题模板 Skill

通过截图即可为 pudding-resume 添加新的简历主题模板。你只需提供一张参考图，AI 会按照 SKILL.md 定义的标准流程完成所有前端和后端改动。

## 🚀 使用方式

1. **截一张你喜欢的简历模板图**
2. **告诉 AI 你想要什么**，例如：
   - "帮我按这张图创建一个新的简历布局"
   - "参考这个设计，做一个类似的简历主题"
   - "把这个模板的效果复刻到项目里"
3. AI 会自动完成**：分析布局类型 → 编写前端布局代码 → 注册到系统 → 补充后端种子数据 → 验证构建

## 🛠️ AI 会做什么

AI 收到你的截图后会按照以下步骤执行：

| 步骤 | 做什么 | 对应文件 |
|------|--------|----------|
| 🔍 分析截图 | 判断布局类型（单栏 / 居中抬头 / 侧栏双栏）、标题样式、配色 | — |
| 🎨 创建布局 | 编写 `LayoutDefinition`，包含 CSS 样式覆盖 | `frontend/src/registry/layouts/<id>.ts` |
| 📋 注册布局 | 将新布局加入布局注册表 | `frontend/src/registry/layouts/index.ts` |
| 🌐 添加翻译 | 补充中文和英文模板名称 | 简历语言文件 |
| 🗄️ 后端种子 | 补充风格库数据和默认配色 | `backend/database/seed.go` |
| ✅ 验证构建 | TypeScript 类型检查 + Vite 构建 + Go 测试 | — |

## ⚠️ 注意事项

- 截图越清晰，AI 还原效果越好
- 如果效果不满意，可以继续对话调整（比如"把标题改成下划线样式"、"换一个配色"）
- 新增的布局会和现有模板共存，不会影响已有布局
